# Platform Layer Checklist

Add platform applications in GitOps sync waves:

1. Gateway API CRDs.
2. Cilium.
3. Load-balancer address pool.
4. cert-manager.
5. Argo CD self-management.
6. Vault and External Secrets.
7. Admission policies.
8. Network policies.
9. Observability agents.
10. Workload apps.

Keep the private operational manifests in a private repo. Public examples should
use placeholders only.
